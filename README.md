# Сайт Maison More

Сайт архитектурной и дизайнерской студии из Беларуси.

## О проекте

Портфолио-сайт для Maison More — студии архитектуры и интерьерного дизайна, специализирующейся на жилых, коммерческих и общественных проектах.

## Технологии

- React 18
- TypeScript
- Vite
- React Router
- Tailwind CSS

## Структура

- **Work** — портфолио проектов (18 проектов)
- **Studio** — информация о студии
- **Process** — процесс работы
- **Contact** — контактная информация

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
